# Emotion Detection Using CNN
This is a real-time facial emotion detection system built using Python, TensorFlow, Keras, and OpenCV. It can detect emotions such as Happy, Sad, Angry, Fear, etc., from a static image or webcam.
## Features
- Trained on FER2013 dataset
- Real-time webcam detection
- Simple CNN model for 7-class classification
